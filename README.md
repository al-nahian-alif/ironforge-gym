# 🏋️ IronForge Gym — Next.js Website

A bold, high-energy gym website built with Next.js 14.

## Setup

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build for Production

```bash
npm run build
npm start
```

## Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Pure CSS with custom properties
- **Images**: Unsplash via next/image
- **Fonts**: Bebas Neue, Barlow Condensed, Rajdhani (Google Fonts)

## Sections

1. **Navbar** — Fixed, transparent → frosted on scroll
2. **Hero** — Full-screen with animated headline and live stats
3. **Ticker** — Scrolling marquee of classes
4. **About** — Image collage + feature cards
5. **Classes** — 6-up grid with hover reveals
6. **Trainers** — 4-card grid with social hover
7. **Pricing** — 3 plans with featured card highlight
8. **CTA** — Full-bleed background call-to-action
9. **Footer** — Links, hours, newsletter signup

## Design

- **Theme**: Industrial dark with electric red accents
- **Fonts**: Bebas Neue (display), Rajdhani (body), Barlow Condensed (hero accents)
- **Colors**: Near-black `#080808`, Electric red `#FF1B1B`, Gold `#F5A623`
- **Clip-paths**: Angular chamfered corners for a hard-edged feel
